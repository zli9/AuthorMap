AuthorMaps

AuthorMaps is a project within the module M-LSI-P-012 Programming Lab 2 of the masters program Life Science Informatics (University of Bonn).
The scope of this project is to provide a search interface for PubMed's API, to determine a list of co-authors for a given author, and visualize co-author connections in a network.

Developed by Jackrite To, Zexin Li, Dhwani Solanki, Sanjana Srinivasan, and Justus Bisten
